# 更新日志 (CHANGELOG.md)

## [1.0.0] - 2026-02-07

### 新增
- **构建流程**: 支持“构建-签名-打包”的分步构建流程。
- **构建脚本**: 
  - `npm run build:dir`: 仅编译并解包，用于生成待签名的文件夹。
  - `npm run build:pack`: 将已签名的文件夹打包为最终安装包。

### 变更
- **输出目录**: 构建输出目录从 `dist_electron_v1.0.0` 调整为 `dist_electron_unpacked` 以支持新的流程。

## [0.1.1] - 2026-01-29

### 修复
- **窗口控制**: 修复了 Windows 平台下右上角最小化、最大化、关闭按钮无法点击的问题。
- **点击拦截**: 解决了登录按钮和顶部广告位上半部分被窗口拖拽层拦截点击的 Bug。
- **布局优化**: 重新设计了顶部 Header 的安全区域，确保业务按钮与系统控制条互不干扰。

### 变更
- **组件化**: 引入 `WindowControls` 独立组件管理窗口行为。
- **安全性**: 在生产环境下禁用了开发者工具 (DevTools)。
- **样式规范**: 在 `index.css` 中标准化了 Electron 拖拽区域的样式定义。
