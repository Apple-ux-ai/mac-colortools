# Locale patches (flat key-value)

This folder contains **245 flat i18n keys** (see `_keys.json`) and generates one patch file per locale: `ar.json`, `bn.json`, `es.json`, … (23 locales).

## Run (single entry point)

From project root or from this folder:

```bash
node scripts/locale-patches/script.cjs
```

- Reads `src/i18n/locales/en.json` and flattens it.
- Uses `translations.json` if present; otherwise builds it from `translations-rows.json` (245 rows × 23 locales) and writes it.
- Writes 23 files: `ar.json`, `bn.json`, `es.json`, `fa.json`, `fr.json`, `he.json`, `hi.json`, `id.json`, `it.json`, `ms.json`, `nl.json`, `pl.json`, `pt.json`, `pt-BR.json`, `ru.json`, `sw.json`, `ta.json`, `th.json`, `fil.json`, `tr.json`, `uk.json`, `ur.json`, `vi.json`.

Each patch file is a single JSON object with exactly those 245 keys in `_keys.json` order; values are the translation for that locale. Placeholders `{{value}}` and `{{count}}` are preserved.

## Regenerating from nested locale files

When you edit `src/i18n/locales/{locale}.json` (e.g. `ar.json`, `fr.json`), regenerate the flat rows and then the patches:

```bash
node scripts/locale-patches/build-rows-from-locales.cjs
node scripts/locale-patches/script.cjs
```

- `build-rows-from-locales.cjs` reads all 23 nested locale files, flattens each, and writes `translations-rows.json` (245 rows × 23 columns).
- `script.cjs` then builds `translations.json` from that (if missing or you deleted it) and writes the 23 patch files.

## Files

| File | Purpose |
|------|--------|
| `_keys.json` | Ordered list of 245 flat keys (same order in every patch). |
| `script.cjs` | Main script: writes all 23 patch JSON files. |
| `build-rows-from-locales.cjs` | Builds `translations-rows.json` from `src/i18n/locales/*.json`. |
| `generate-translations.cjs` | Alternative: builds `translations.json` from `translations-rows.json`. |
| `translations-rows.json` | 245 rows × 23 locale columns (generated by build-rows-from-locales.cjs). |
| `translations.json` | `{ "ar": { "key": "..." }, ... }` (built by script.cjs from rows when missing). |
| `ar.json` … `vi.json` | 23 flat patch files (output of script.cjs). |
